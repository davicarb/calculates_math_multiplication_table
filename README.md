# calculates_math_multiplication_table
This simple application was built without the use of AI, with the purpose of training logic thinking and problem solving. One of my first projects ever built using C#, so it may me simple, but useful.
One of the goals here is to train logic and the use of for, while and do while loops.
My goal is to be one day a developer for a big tech company outside of my home country Brazil.

I created the application using three variables. One for the do while loop (option), and two for doing the math calculations (constantNumber) and (variableNumber).

The variable "Option" is of a string type, just so that when the user inputs "Y" or "y" (which stands for "YES") at the end of the program, the program repeats again.
It's outside of the do while loop, just so that when the conditional is asked at the end of the program, the compiler "finds" the variable.

Inside the do while loop it's the working program.

I named the variables constantNumber and variableNumber so it's easy to visualize how it works.
The user inputs the constantNumber (the one that will be multiplied by the variableNumber, which will be floating between 0 and 10 while the loop is running).

And the variableNumber the user does not input. It is determined, by the for loop, that it will start on 0, and will end on 10.
So, when the loop starts, it always multiplies the inputed number for the number that is constant (0 to 10), which is done inside the for loop.
